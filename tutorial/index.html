Kafka Stateful Stream Processor

This is an activator project showcasing stateless stream processing using Kafka KStream.

Here we are using:

Kafka Client for Kafka API
Twitter4J Streaming as a source.
Typesafe Config to read configuration file.
Kafka Streaming to process stream

Steps to Install and Run Zookeeper and Kafka on your system :

Step 1: Download Kafka
Download Kafka from here
Step 2: Extract downloaded file

        tar -xzvf kafka_2.11-0.10.1.1.tgz
        cd kafka_2.11-0.10.1.1
Step 3: Start Servers
Start Zookeeper:

        bin/zookeeper-server-start.sh config/zookeeper.properties
Start Kafka server: 

        bin/kafka-server-start.sh config/server.properties

Clone Project

git clone https://github.com/knoldus/kafka-stateless-kstream.git

Start Tweets Producer using,


        bin/activator "run-main com.knoldus.demo.TweetProducer"
This will start fetching the tweets and push each tweet into a Kafka topic queue.

Start Stream Processor using,


        bin/activator "run-main com.knoldus.demo.KStreamProcessor"
This will start stream processing on the tweets.

Start Consumer using,


        bin/activator "run-main com.knoldus.demo.KStreamConsumer"
This will start fetching all the hashTags in each tweet.
